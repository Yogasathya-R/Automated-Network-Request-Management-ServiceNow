# Automated Network Request Management in ServiceNow

## 1. Introduction
The application uses ServiceNow Service Catalog, Flow Designer, approvals, and a custom database table to manage the complete request lifecycle.

## 2. Project Objectives
- Implement approval-based workflows
- Track request status and assignments
- Improve operational efficiency and reduce manual errors

## 3. Scope of the Project
The project covers the complete network request lifecycle:
- Request submission through the Service Portal
- Approval by authorized users
- Automatic assignment to support teams
- Status tracking and audit history maintenance

## 4. System Architecture
The solution is built using the following ServiceNow components:
- **Service Portal** – User interface for request submission  
- **Service Catalog** – Network Request catalog item  
- **Flow Designer** – Approvals and assignments  
- **Custom Network Database Table** – Stores request information  
- **Notifications and Audit History** – Tracks updates and changes  

## 5. Project Phases Overview

### Ideation Phase
This phase focuses on identifying the problem, understanding user needs, and defining project objectives. Stakeholder mapping and problem statements are documented.

### Planning Phase
Functional requirements, workflow steps, and approval logic are planned in this phase.

### Project Design
Includes catalog item creation, form and variable configuration, and Flow Designer setup.

### Performance Testing
Ensures correct request submission, approval flow, state transitions, and notifications.

## 6. Key Features
- User-friendly request submission
- Real-time request status tracking
- Centralized request data management

## 7. Tools & Technologies Used
- **Platform:** ServiceNow  
- **Modules:** Service Catalog, Flow Designer, Service Portal  
- **Database:** Custom Network Database Table  

## 8. Conclusion
The Automated Network Request Management project demonstrates how ServiceNow can be used to efficiently automate network service requests. The solution improves visibility, accountability, and operational efficiency while ensuring a structured and auditable process.
